# 易兌-論子平 對齊架構

依 易兌 論八字排盤邏輯對系統進行校準。

## 參考案例

### 案例 B：李羿函
- **性別**：陽 坤造（女）
- **西曆**：1994-06-20 23:26
- **農曆**：甲戌年五月十二 晚子時
- **四柱**：甲戌 庚午 丁丑 壬子
- **大運**：1999己巳、2009戊辰、2019丁卯…（起運 4年10月，虛歲6歲起；6–15、16–25…）
- **晚子時**：日柱用當日（6/20→丁丑），非次日

### 案例 A：陳弘林
- **姓名**：陳弘林
- **性別**：陰乾造（男）
- **西曆**：1983-08-25 14:55
- **農曆**：癸亥年七月十七 未時
- **地理經度**：120.288°E（台南）

## 1. 真太陽時

- **鐘錶時間**：14:55
- **真太陽時**：14:53（約早 2 分鐘）
- **實作**：採用 `true-solar-time.js` 的 equation of time，不再只做經度簡化
- **台南經度**：120.288（與易兌一致）

## 3. 大運

- 1989 己未、1999 戊午、2009 丁巳、2019 丙辰
- 2029 乙卯、2039 甲寅、2049 癸丑、2059 壬子
- **起運**：出生後 5 年 8 個月（虛歲 7 歲）
- **節氣**：立秋 1983-08-08 10:29（台灣）

## 4. 空亡

依各柱干支所屬旬分別計算：
- 癸亥、庚申 → 甲寅旬 → 空子丑
- 乙酉 → 甲申旬 → 空午未
- 癸未 → 甲戌旬 → 空申酉

## 5. 自坐

各柱地支對該柱天干的十二長生（非僅對日主）。

## 6. 日柱曆法

盲派／易兌模式：`DAY_PILLAR_OFFSET = 10`，1983-08-25 → 乙酉。
