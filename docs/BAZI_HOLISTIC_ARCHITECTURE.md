# 🧬 八字命理全息分析架構（精微版）

本架構旨在超越基礎標籤，透過 **五行生態、十神情境、宮位互動、動態心流** 四個維度，揭示命局中幽微的矛盾、天賦與人生劇本的獨特細節。

對應實作：`js/bazi-system.js` 之 `BaziAnalyzer`（全局解碼、個性、事業、財運、感情婚姻分析）。

---

## 第一步：全局解碼——定盤與生態分析

**目標**：不再僅滿足於「身強身弱」，而是解讀命局的「能量生態」。

### 能量氣象

- **寒暖燥濕**：不僅看調候用神，更看其造就的**心性底色**。如全局水寒，性喜靜多思；木火炎燥，性急智敏。
- **清濁成敗**：看干支組合是純而不雜（清），還是交戰刑沖（濁）。清者性專，濁者心雜事繁。

### 五行勢力圖

- 繪製五行力量分布，找出「**最旺五行**」（勢能）與「**最弱／缺失五行**」（勢缺）。
- **關鍵**：分析旺神是**順用**（食傷、財、官）還是**逆用**（比劫、梟印），決定格局高低。

## 第二步：個性分析架構——心性的多維光譜

**目標**：描繪一個活生生的人，而非一組形容詞。

| 分析維度 | 精微解析要點 | 舉例與深化 |
|----------|--------------|------------|
| **1. 心性本源** | 日主五行 + 月令氣象 | 秋月庚金：帶煞的刀劍（剛毅）；夏月癸水：及時雨（靈動機敏）。 |
| **2. 思維模式** | 印星與食傷的制化關係 | 印克食傷：思想被傳統約束；食傷配印：才華有學識支撐；無印食傷旺：天馬行空但流於浮誇。 |
| **3. 行為驅力** | 官殺與日主的親和度 | 官印相生：壓力化為責任；殺刃兩停：魄力抗壓；官殺混雜無制：規則與叛逆衝突。 |
| **4. 情感與價值觀** | 財星與比劫的配置 | 財透比劫藏：外表大方內心計較；比劫旺無財：重情義輕財物；財星合印：金錢投入學習／精神。 |
| **5. 隱藏面具** | 天干透出 vs 地支藏干 | 天干正官、地支藏七殺傷官 → 外在循規、內心叛逆的「雙重人格」配置。 |

## 第三步：事業分析架構——天賦、舞台與節奏

**目標**：區分「能做的」、「適合做的」與「命運推動去做的」。

| 分析維度 | 精微解析要點 | 舉例與深化 |
|----------|--------------|------------|
| **1. 天賦領域** | 食傷與祿神的方位 | 食神在月柱：家族技藝傳承；在時柱：晚年靠技藝或學生、作品成名。 |
| **2. 成就舞台** | 官殺與印星的虛實 | 虛官透干：好名聲、虛職；實殺坐支：實權、硬指標壓力；印星為喜貼身：大型平台、機構、國企。 |
| **3. 行業軌跡** | 十神組合的象法拓展 | 庚金七殺：軍警／外科／金融風控；壬水偏印：玄學、心理、跨境貿易。 |
| **4. 事業節奏** | 大運對原局的關鍵解鎖 | 原局財官旺身弱：早年助身則少年得志、中年助身則大器晚成；食傷被印：大運逢財破印為「解凍」之運。 |

## 第四步：財運分析架構——財源、庫鑰與流轉

**目標**：分析財富的「來路、去路與存量」。

| 分析維度 | 精微解析要點 | 舉例與深化 |
|----------|--------------|------------|
| **1. 財星質量** | 財星的坐基與護衛 | 財坐比劫：錢左手進右手出；財坐官殺：來自官方或壓力行業，能存但辛苦；財星被合：為情緣花錢或被合走。 |
| **2. 得財手段** | 生財路徑的十神組合 | 傷官生財：創新、投機、高風險高回報；食神制殺生財：專業解難（工程師、律師）；比劫奪財：體力、團隊分紅、競爭性銷售。 |
| **3. 財富周期** | 財庫（辰戌丑未）的開合 | 原局有財庫：善守財；大運流年沖開：爆發性收入或破財；財庫逢合：財富鎖住或轉為固定資產。 |
| **4. 消費傾向** | 財星所生所克的對象 | 財生官殺：花錢買地位、名聲或為配偶／子女；財破印：花錢學習、買房或因物質損名譽。 |

## 第五步：感情婚姻分析架構——緣起、質變與因果

**目標**：解析關係的本質模式、吸引力來源與潛在危機。

| 分析維度 | 精微解析要點 | 舉例與深化 |
|----------|--------------|------------|
| **1. 吸引力原型** | 配偶星與桃花星的特質 | 偏財／七殺：特立獨行、侵略性／神秘感；正官／正財：端莊、穩定、社會評價高；桃花在劫財：豪爽義氣、哥們型伴侶。 |
| **2. 關係模式** | 夫妻宮與配偶星的互動 | 宮星同位：夫妻一體、觀念相近；宮星相克：觀點衝突、互相制約；配偶星入墓於夫妻宮：保護欲或控制欲強、關係封閉。 |
| **3. 婚姻穩定性** | 刑沖合害的深度解讀 | 夫妻宮逢合：配偶易有外情或人際影響婚姻；夫妻宮逢刑：互相傷害難分離；配偶星被比劫合走：第三者爭奪之象。 |
| **4. 婚緣時機** | 引動婚姻的複合條件 | 流年配偶星 + 合入夫妻宮／日主：強烈結婚信號；流年沖動夫妻宮 + 解除不利：變動中成婚；大運引動時柱：奉子成婚。 |

## 💎 架構心法：從「靜態標籤」到「動態敘事」

- **矛盾即真相**：命局中最突出的矛盾點（如印旺傷官也旺）正是人生核心課題，也是個性最獨特之處。
- **位置定層次**：同一十神在年柱（社會、長輩）、月柱（事業、自我）、日柱（婚姻、內心）、時柱（晚年、子女）作用完全不同。
- **合沖定事件**：「合」代表緣分、合作、得到；「沖」代表變動、衝突、失去。是人生情節推進的關鍵節點。
- **歲運定應期**：原局是劇本，大運是章節，流年是具體場景與台詞。分析須三層聯動，方能斷事如神。

此架構致力於將八字還原為**有溫度、有矛盾、有起承轉合的生命故事**。
