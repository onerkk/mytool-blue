# 三才配置分析核心架構（純淨版）

本架構僅保留判斷邏輯本身，可作為算法基礎或分析思維導圖直接應用。對應實作：`js/nameology-system.js` 之五格計算、三才生克與三維度評價。

> **完整版**：基礎計算鐵律、外格公式、三才吉凶、格位維度與八字聯動法則見 **`NAMEOLOGY_CALCULATION_ARCHITECTURE.md`**。

## 一、基礎定義與計算規則

### 五格劃分

| 格位 | 計算規則 |
|------|----------|
| **天格** | 姓氏筆畫數 + 1（複姓取兩字和） |
| **人格** | 姓氏末字 + 名字首字筆畫和 |
| **地格** | 名字全部筆畫和（單名 + 1） |
| **總格** | 姓名總筆畫數 |
| **外格** | 總格 − 人格 + 調整數 |

### 五行轉換規則（尾數）

| 尾數 | 五行 |
|------|------|
| 1、2 | 木 |
| 3、4 | 火 |
| 5、6、0 | 土 |
| 7、8 | 金 |
| 9、0 | 水 |

註：尾數 0 依流派可歸土或水；本架構採用「尾數 0 → 土」，與 5、6 同屬土，以利算法一致。

## 二、三才生克關係矩陣

天格 → 人格 → 地格 的五行生克決定基礎格局：

| 生克類型 | 能量模式 | 核心特徵 |
|----------|----------|----------|
| **相生鏈**（如金→水→木） | 順生流通 | 運勢通達，內外和諧 |
| **逆生鏈**（如木→火→土） | 反哺滋養 | 根基穩固，晚運豐隆 |
| **比和局**（如土→土→土） | 能量聚焦 | 專精深耕，易失變通 |
| **受克局**（人格被克） | 壓力制約 | 抗壓成長，先難後易 |
| **發散局**（人格克他） | 主導消耗 | 開拓進取，易顯疲憊 |
| **戰克局**（多行相戰） | 內在衝突 | 多線程應對，潛力激發 |

## 三、格位象徵意義

| 格位 | 象徵 |
|------|------|
| **人格（核心主體）** | 主導：中年運、核心性格、自我認知；五行狀態決定行為模式底色。 |
| **地格（基礎環境）** | 主導：前半生、家庭根基、內在安全感；與人格關係決定內外一致性。 |
| **天格（先天條件）** | 象徵：祖蔭、時代機遇、不可控外力；對人格生克決定「得天助」程度。 |
| **總格（最終導向）** | 反映：人生總體成就、晚年狀態；需觀其與人格的五行呼應關係。 |
| **外格（互動介面）** | 表徵：社交面具、突發機遇、副業潛能。 |

## 四、分析決策樹

```
開始
  ↓
計算五格五行
  ↓
提取【天-人-地】三才
  ↓
判斷生克關係類型
  ↓
分析人格狀態：
  1. 是否得生（天格生之）？
  2. 是否得助（地格比和）？
  3. 是否受壓（被天/地克）？
  ↓
評估能量流通性：
  - 順生：氣暢
  - 逆生：氣聚
  - 戰克：氣雜
  - 受制：氣滯
  ↓
檢查關鍵節點：
  1. 人格五行屬性強度
  2. 地格對人格的承載度
  3. 總格對人生終局的導向
  ↓
輸出三維度評價：
  ① 穩定性（地格支持度）
  ② 發展性（天格助力度）
  ③ 協調性（三才流通度）
```

## 五、核心心法要點

- **人格為體**：一切分析以人格五行為中心點。
- **生克為用**：重視五行間的動態作用，而非靜態吉凶。
- **流通為貴**：連續相生 ＞ 孤立吉格 ＞ 交戰格局。
- **平衡為要**：避免單一五行過旺或過弱。
- **導向為終**：總格五行應與人、地二格有邏輯呼應。

此架構剝離所有實例與延伸，僅保留判斷邏輯本身，可直接作為算法基礎或分析思維導圖應用。
