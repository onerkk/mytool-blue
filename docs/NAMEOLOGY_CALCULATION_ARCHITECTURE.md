# 姓名學基礎計算鐵律與三才八字聯動架構

本架構為純淨的「輸入－處理－輸出」邏輯鏈。所有感性描述與吉凶斷語，均須源於此鏈中五行生克的客觀推演，並與八字喜忌嚴格對應，方能稱為嚴謹的姓名學分析。對應實作：`js/nameology-system.js`。

---

## 📋 負責任的姓名學分析：完整四步鏈條

一個負責任的姓名學分析，**必須**依序完成以下完整鏈條：

| 步驟 | 內容 | 說明 |
|------|------|------|
| **第一步：輸入校準** | 輸入標準漢字姓名；輸入使用者**精準生辰**（用於排八字定喜忌）。 | 缺一則無法進行第四步裁定。 |
| **第二步：計算與轉換** | 按規範計算姓名各字**筆畫**；依公式得出**五格數理**；將數理個位數轉換為**五行**。 | 筆畫計算模組須絕對準確，否則結論無參考價值。 |
| **第三步：格局分析** | 提取天、人、地**三才五行**，分析生克關係；評估各格數理本身的傳統吉凶含義（如 23 數「旭日東升」）。 | 僅完成此步時，無法判斷姓名是否適用於命主。 |
| **第四步：與八字聯動（決定性步驟）** | 排出使用者八字，確定**喜用神與忌神**；將姓名**五格五行**與**八字喜忌**逐一比對；核心判斷：人格、總格五行是否為喜用神？三才流通終點是否指向喜用神？ | **八字喜用神分析為姓名學分析的必填前置步驟。** |

### 最終裁定（僅在完成第四步後有意義）

| 裁定 | 條件 |
|------|------|
| **吉名** | 五行補益喜用，且三才生克和諧。 |
| **凶名** | 五行加強忌神，即便三才生克吉亦凶。 |
| **平名** | 五行不助忌神，但補益力弱。 |

**結論**：若僅完成第三步（格局分析），產出如同「製作精良的汽車部件」——在不知道要安裝到什麼型號的汽車（八字）上之前，無法判斷其是否適用。系統**務必**將八字喜用神分析設為姓名學聯動裁定的**必填前置步驟**，並確保筆畫計算模組絕對準確，如此產出的結論方有參考價值。

## 🧮 第一部分：基礎計算鐵律（輸入校準）

此為一切分析之絕對前提，錯則全盤皆錯。

### 筆畫計算規則

| 項目 | 說明 |
|------|------|
| **字體** | 以《康熙字典》繁體字為標準，或明確約定某一字庫（如《姓名學大辭典》）。 |
| **部首** | 按正確部首計畫（如「阝」為 7 畫，「⻌」為 7 畫，「氵」為 4 畫）。 |
| **特殊字形** | 固定特殊字形筆畫（如「艹」為 6 畫）。 |

**校準範例（筆畫僅作公式說明，不涉及真實個資）**：若姓氏 16 畫（部首「阝」以 7 畫計）、名一 5 畫、名二 8 畫 → 天格 17 金（姓+1）、人格 21 木（姓+名一）、地格 13 火（名一+名二，名字全部筆畫和）、外格 9 水（名二+1）、總格 29 水。三才 **金→木→火**。

### 五格數理公式

| 格位 | 計算規則 |
|------|----------|
| **天格** | 單姓：姓氏筆畫 + 1；複姓：姓氏兩字筆畫之和。 |
| **人格** | 姓氏末字筆畫 + 名字首字筆畫。 |
| **地格** | **標準五格**：單名 名字筆畫 + 1；雙名 **名字全部筆畫和**（兩字或以上皆取全部和），以與主流姓名學理論一致，避免誤導性結論。 |
| **總格** | 姓氏與名字全部筆畫總和。 |
| **外格** | 單姓單名：2；單姓雙名：名字末字筆畫 + 1；複姓單名：姓氏首字筆畫 + 1；複姓雙名：姓氏首字筆畫 + 名字末字筆畫。 |

### 五行轉換規則

數理尾數與五行對應關係恆定：

| 尾數 | 五行 |
|------|------|
| 1、2 | 木 |
| 3、4 | 火 |
| 5、6、0 | 土 |
| 7、8 | 金 |
| 9、0 | 水 |

註：尾數 0 通常計為 10，屬**土**（部分流派亦作水，需統一）。本架構採用「尾數 0 → 土」。

## ⚙️ 第二部分：三才生克核心矩陣（關係判定）

「三才」指 **天格、人格、地格** 三者之五行關係。此為分析中樞。

| 生克關係類型 | 五行生克邏輯 | 核心能量解讀 | 吉凶 |
|--------------|--------------|--------------|------|
| **順生** | 天→人→地 連續相生（例：金生水，水生木） | 氣運流通，內外和諧，順勢而成，多得助力。 | 吉 |
| **逆生** | 地→人→天 逆向相生（例：木生火，火生土） | 根基滋養，厚積薄發，內在能量反哺外在成就。 | 中吉 |
| **比和** | 天=人=地 三者相同（例：土土土） | 能量聚焦，專一性強，易在特定領域深耕，但失之變通。 | 中吉 |
| **人格受生** | 天→人 生 或 地→人 生 | 得助：外界（天）或基礎（地）生扶自身，易得機遇或內心安定。 | 吉 |
| **人格克出** | 人→天 克 或 人→地 克 | 主導：主動克服環境壓力，具開拓性，但也意味著付出與消耗。 | 中平 |
| **人格受克** | 天→人 克 或 地→人 克 | 受壓：外界壓力大或內心壓抑，需在克制中成長，事倍功半。 | 凶 |
| **戰克** | 天、人、地 多行相戰（例：金木土，木克土、金克木） | 內在衝突：多線程應對矛盾，人生挑戰多，潛力與壓力並存。 | 凶 |

## 🔗 第四部分：與八字聯動法則（終極驗證）

此為最高層應用，決定姓名能量是否真正有益於命主。

### 第一步：八字定盤

精確排出八字，確定 **日主旺衰** 與 **喜用神、忌神**。

### 第二步：五行映射

將姓名五格之五行，分別列出。

### 第三步：戰略比對

- **核心優先**：人格、總格的五行，必須優先為八字 **喜用神**。
- **三才氣韻**：三才配置的五行生克結果，應形成「最終生助人格（喜用神）」的流向。
- **禁忌檢查**：確保姓名五行不強化八字 **忌神**，尤其忌神成為人格時，為大凶。

### 第四步：綜合裁定

| 裁定（對外用語） | 內部對應 | 條件 |
|------------------|----------|------|
| **吉名** | 上佳／可用 | 五行補益喜用，且三才生克和諧。 |
| **凶名** | 禁用 | 五行加強忌神，即便三才生克吉亦凶。 |
| **平名** | 慎用 | 五行不助忌神，但補益力弱。 |

此架構確保所有吉凶斷語源於五行生克的客觀推演，並與八字喜忌嚴格對應。

## 📌 必要聯動與免責聲明

### 規則建議

- **地格**：堅持使用 **標準五格計算**（地格 = 名字全部筆畫和），以與主流姓名學理論相容，避免產生誤導性結論。
- **八字聯動**：姓名吉凶的最終判斷 **必須** 與使用者的 **八字喜用神** 結合。例如：
  - 若八字 **喜木**，則此名人格木（雖受克）可為用，但需付出更大努力。
  - 若八字 **忌木**，則此名人格木為忌，三才克戰會加劇人生矛盾，凶象顯著。

### 輸出時務必強調

系統在輸出分析時，**務必** 明確標示以下免責聲明，以確保專業性與準確性：

> **「姓名吉凶的最終判斷必須與使用者的八字喜用神結合，請綜合判斷。」**

本分析僅供參考，不取代專業命理建議。

## 📌 聯動分析參考（範例：某姓名 × 八字喜水木）

**核驗**：八字（範例：某男命）日主乙木、申月官殺旺、身弱；喜用神水（印）、木（比劫），忌神金、土、火。若姓名五格為 天格 17 金、人格 21 木、地格 13 火、外格 9 水、總格 29 水，三才 **金→木→火**。

**聯動要點**：人格木為八字喜神→大吉，能直接助身抗殺；總格水為用神→吉，晚年與總體成就得滋養。三才中天格金克人格木（外界壓力）、人格木生地格火（能量消耗），形成「喜神受克、先難後成」。姓名核心（人格、總格）精準補益喜用，屬關鍵性補益；綜合為中等偏上、此名可用，過程波折但終能受益。
