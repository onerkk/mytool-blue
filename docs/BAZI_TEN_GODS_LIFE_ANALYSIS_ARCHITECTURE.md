# 八字十神與人生領域分析完整架構

本文件對應「個性 → 事業 → 財運 → 桃花婚姻」四大領域的分析邏輯，與前端 `bazi-system.js` 中 `BaziAnalyzer` 的 `analyzePersonality`、`analyzeCareer`、`analyzeWealth`、`analyzeRelationship` 及相關十神／神煞計算。

---

## 架構使用心法（總則）

1. **先定喜忌，再論事項**：一切分析的首要前提是明確命局的**喜用神與忌神**。同一顆財星，是喜神則論富，是忌神則論貧。
2. **宮位星位，結合參看**：日支（夫妻宮）看配偶本質，財官星（配偶星）看配偶緣分，二者要結合。
3. **大運流年，應期關鍵**：命局是車，大運是路，流年是天氣。好車遇上好路好天氣，方能一路順風。
4. **平衡中庸，避免極端**：分析時需兼顧正反兩面，任何特徵都有其優勢與挑戰。

## 第二步：事業分析框架（以官殺、印星、財星、食傷與喜忌為核心）

### 事業類型定位

- **官殺為主導**：適合管理、公職、軍警、紀律性強、有社會地位的行業。
- **印星為主導**：適合文化、教育、研究、諮詢、公益、穩定機構。
- **財星為主導**：適合金融、貿易、商業、銷售、實業經營。
- **食傷為主導**：適合技術、藝術、創作、演藝、策劃、自由職業。
- **比劫為主導**：適合合作創業、體育競技、需要體力的行業。

### 發展層次判斷

- **看喜用神**：
  - 事業星（官殺／印星／財星／食傷）為喜用且有力 → 在該領域天賦高、成就大。
  - 事業星為忌神但有制化 → 能駕馭壓力，將挑戰轉化為成就。
  - 事業星為忌神無制化 → 事業多阻、壓力重重，需選擇其他方向。
- **看調候**：
  - 命局寒（冬生）需火調候，從事火相關行業（能源、互聯網）更利。
  - 命局燥（夏生）需水調候，從事水相關行業（物流、貿易）更利。

### 運勢與時機

- 大運或流年引動**喜用神**的事業星 → 升職、創業、轉行成功的關鍵期。
- 大運或流年引動**忌神**的事業星 → 職業瓶頸、壓力倍增、決策易錯的時期。

**分析邏輯**：先定**行業類型**，再辨**格局高低**，後抓**發展時機**。

## 第三步：財運分析框架（以財星、食傷、比劫與宮位為核心）

### 財源模式

- **正財**：穩定收入、工資薪金、合法經營所得。宜**穩中求財**。
- **偏財**：意外之財、投機收入、風險投資、大額貿易。宜**動中求財**。
- **食傷生財**：以技術、才智、口才、創意生財。多為**主動求財**。
- **官殺護財**：以權力、地位、管理能力生財或守護財富。
- **比劫奪財**：易因朋友、合作、競爭破財，或需要共享財富。

### 財富層次

- **身強財旺**：能擔大財，富命。
- **身弱財旺**：富屋貧人，求財辛苦，需行幫身運方能發財。
- **財星為喜用**：財富是人生助力，得財順遂。
- **財星為忌神**：財富是人生煩惱之源，易因財致禍。
- **財庫（辰戌丑未）**：有財庫者能守財，逢沖則開庫有發大財機緣。

### 求財時機與風險

- 大運或流年**財星為喜用神且被引動** → 財運高峰期。
- 大運或流年出現**比劫奪財**組合（尤其為忌） → 破財高峰期。

### 完整財運分析四維度（架構實例）

| 維度 | 內容 | 實例（甲戌 庚午 戊寅 壬子） |
|------|------|-----------------------------|
| **1. 財富本源** | 財星強度、位置、生存環境（是否逢沖） | 時干透壬水偏財，時支子水為財根；子午沖財根不穩，為財務波動根源。 |
| **2. 得財方式** | 財星與日主之間的橋樑十神 | 食神生財（庚→壬）：憑專業、口才、創意獲財；七殺攻身：亦可從管理、高壓行業得財。 |
| **3. 財富層次** | 身強弱能否擔財 | 戊土身強，足以勝任旺財，身強財旺之局，層次中上。 |
| **4. 得失時機** | 大運流年對財星的激活與損傷 | 得財：金水旺運（如 44 歲後乙丑、甲子、癸亥）及猴鼠雞年。破財：子午沖之馬鼠年、土旺年慎防合作耗財。 |

**分析邏輯**：先明**財富本源**，再看**得財方式**，後判**財富層次**，最後察**得失時機**。

## 第四步：桃花與婚姻分析框架（以夫妻宮、官殺、財星、十神組合為核心）

### 婚姻基礎（夫妻宮與星）

- **夫妻宮（日支）**：配偶的個性、家庭狀況及婚姻內在感受。
  - **日支為喜用**：配偶對自己有幫助，關係和諧。
  - **日支為忌神**：配偶條件或關係易成壓力源。
- **配偶星**：**男命看財星**，**女命看官殺**。
  - **星宮皆喜用**：婚姻質量高。
  - **星宮相戰**（如男命日支坐比劫奪財）：婚姻易有衝突。

### 桃花特質與機遇

- **正官／正財**：正緣，易遇穩定、傳統的對象。
- **七殺／偏財**：偏緣，易遇熱烈、短暫或條件懸殊的戀情。
- **桃花星（子午卯酉）**：異性緣佳，魅力強。
- **紅鸞、天喜**：婚戀喜慶之事。
- **配偶星出現的大運與流年**：是結識對象、談婚論嫁的關鍵年份。

### 婚姻穩定性

- **夫妻宮逢沖**（如子午沖）：婚姻關係不穩定，易有變動。
- **夫妻宮逢合**（如寅亥合）：配偶易有外情，或自己人緣過好。
- **配偶星混雜**（正偏俱現）：感情選擇多，易有複雜關係。
- **比劫旺而克配偶星**：婚姻易有競爭者介入。

**分析邏輯**：先看**家宅（宮位）**，再觀**來客（配偶星）**，後查**宅基穩固與否（刑沖合會）**。

## 與「定格→明病→察運→斷事」的銜接

- **喜忌**：來自 `calculateFavorableElements`（調候 > 通關 > 扶抑），為各領域「論喜論忌」的統一前提。
- **大運流年應期**：來自 `dayun-calculator.js`、`_tagDayunWithFavorable` 與 `scoring-engine.js` 的流年評分，用於事業／財運／感情之「關鍵年份」判斷。

此架構提供從八字命盤到人生具體領域的完整解碼地圖，按此步驟結合具體命盤即可進行系統、深入的推導。
