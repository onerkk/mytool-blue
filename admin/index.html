⏰ Prompt Cache TTL 省錢關鍵

控制 system prompt cache 的存活時間。直接影響 cache write 成本。
5m：cache write 1x 價（Sonnet $3.75/M、Opus $6.25/M）—— Anthropic 預設
1h：cache write 2x 價（Sonnet $6/M、Opus $10/M）—— 需 beta header

⚠️ 何時該選 1h：同 1 小時內跑 3+ 次相同 system prompt 的讀盤才划算
💰 何時該選 5m：日流量 <30 次讀盤（靜月之光目前階段）

📊 靜月之光 2026-04-24 實測數據：
三次 Sonnet/Opus 讀盤用 1h TTL 總成本 $3.19 ≈ NT$102
其中 90% 是 cache write 費用（因為沒有 1 小時內多次請求可分攤）
改 5m TTL 預期降至 $2.00 ≈ NT$64（省 37%）
