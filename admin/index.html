💬 追問設定

控制每筆解讀綁幾次免費追問（可再用 用戶卡片 ✏️ 覆蓋個人值），以及追問時送給 AI 的 7 個訓練模組。

每筆解讀綁 N 次免費追問 設 0 = 停用免費追問（所有人直接 NT$19/次）
追問訓練模組（打勾 = 追問時注入到 system prompt）
全部關閉 = 追問只吃基本 prompt 不加情境訓練（最省 token 但品質下降）
全部打開 = 追問最完整（預設值；+7 個模組約 15-20K tokens）
