# 八字全域 UI 修正報告 v1.1.1

## 問題

八字全域分析的自訂日期選擇器雖已取代 Android 原生日曆，但日期格仍使用 `<button>` 元素，且日期格沒有完整重設瀏覽器按鈕外觀。Samsung Internet／Android Chromium 可能套入原生白底、邊框與內距，造成日期全部顯示成白色方塊，與原版單盤 UI 不一致。

## 根因修正

1. 日期格、年月快選、時、分與城市選項改成與原版單盤相同的非原生按鈕格（`div[role="button"]`）。
2. 日期格明確設定透明背景、零邊框、零內距、零陰影及 `appearance:none`，避免任何全域 CSS 或瀏覽器 UA 樣式滲入。
3. 上／下月、取消與確定仍保留語意化 `<button>`，但在選擇器範圍內完整重設 appearance、margin、字型與陰影。
4. 增加 Enter／Space 鍵操作，保留鍵盤可用性。
5. 更新快取版本為 `bazi-suite.js?v=20260626v1_1_1`。

## 套用範圍

- 單人分析
- 合盤 A／B 方
- 人格卡
- 真太陽時工具
- 日期、時間與地區選擇器
